Space Invaders


This project marks a personal milestone as the first major programming endeavor and the very first video game I've ever created. Having played video games my entire life, the experience of transitioning from a player to a developer was both exciting and deeply meaningful. Crafting gameplay mechanics, designing features like dynamic alien movement, and building a polished experience from scratch gave me a profound appreciation for the creative and technical work behind games. Now, I can proudly say I've programmed my own video game—a dream come true for someone who grew up immersed in gaming.

EasyRecipes


EasyRecipes was my first venture into app development, and it served as an exciting and rewarding learning experience. The app allows users to enter and save their own recipes, complete with photos to make each dish visually appealing. Its clean and intuitive interface makes it easy to organize and access recipes, effectively functioning as a personalized digital recipe book.

This project introduced me to core development concepts, from creating user-friendly forms to managing data storage for saved recipes. It was a fun and inspiring way to combine my technical skills with a practical, everyday application.

Infinite Matrix


The Infinite Matrix game was my first introduction into Unreal Engine 5, a powerful and versatile game engine that allowed users to create immersive and engaging gameplay experiences. When I would see the scale of video games that were being made with Unreal Engine, I would always think that it was a level of game development that was years beyond my capabilities. I was extremely intmidated when my Game Development professor mentioned that we would be developing a game in Unreal Engine. After a few tutorials, it very quickly became clear to me that Unreal Engine was not some far-away concept that I would never be able to grasp. Even though this game is a very simple and basic game where you move through an infinite tube making sure not to run into the different shapes, I opened a door for me that I did not expect to have opened for me when I first registered for the course.
